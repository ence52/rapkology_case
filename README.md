# RAP Kology

This is a Next.js 15 project for RAP Kology, a Turkish rap news, music, and video platform. The project features:

- Latest Turkish rap news, music videos, interviews, and trending tracks
- Blog section with dynamic routing for posts
- Hero slider, trends, favorites, and explore sections
- Responsive design using Tailwind CSS
- Newsletter subscription and social media links

## Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Run the development server:**

   ```bash
   npm run dev
   ```

   The app will be available at [http://localhost:3000](http://localhost:3000).

3. **Build for production:**

   ```bash
   npm run build
   npm start
   ```

## Project Structure

- `src/app/` – Main app pages and layouts
- `src/components/` – UI components (Hero, Trends, Favorites, Explore, Blog, etc.)
- `src/data/` – Static data for posts, tags, navigation, etc.
- `src/utils/` – Utility functions
- `public/` – Static assets (images, icons, etc.)

## Technologies Used

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript
- Swiper (for sliders)

## Customization

- Edit content in `src/data/` for posts, tags, and navigation
- Update images in the `public/` directory
- Modify styles in `src/app/globals.css` and Tailwind config
